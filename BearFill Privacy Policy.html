Privacy Policy for BearFill

Last Updated: 03.23.2025

1. Information We Collect

BearFill stores form data to enhance user experience and improve efficiency. The data collected includes:

  • Form field values saved for autofill purposes
  • User preferences related to form completion

2. How We Use the Information

The collected data is used solely for user convenience and form-filling automation. We do not use the data for any other purposes.

3. Data Sharing

We do not share, sell, or distribute any stored user data to third parties.

4. Data Security

We take appropriate security measures to protect stored data. However, BearFill does not collect personally identifiable information beyond what is necessary for form-filling purposes.

5. User Control

Users have full control over their stored data and can delete saved form data at any time through the extension settings.

6. Changes to This Policy

We may update this Privacy Policy periodically. Any changes will be reflected on this page.
